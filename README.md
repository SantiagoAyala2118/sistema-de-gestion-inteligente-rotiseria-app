# Sistema de Gestión de Inventario Inteligente

Sistema web de gestión de ventas, stock e informes para rotisería.
Desarrollado como proyecto profesional individual.

## Stack tecnológico

- **Frontend:** React + TypeScript + Vite
- **Backend:** Node.js + Express + TypeScript
- **Base de datos:** PostgreSQL (Supabase)
- **ORM:** Prisma
- **Deploy:** Railway (backend) + Vercel (frontend)

## Cómo correr el proyecto localmente

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## Variables de entorno

Copiá `server/.env.example` a `server/.env` y completá los valores necesarios.
